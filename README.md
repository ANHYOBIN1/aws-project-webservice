# aws-project-webservice
------

## 사용기술

- JAVA 1.8
- Spring framework
- SpringBoot 2.1.7 RELEASE
- JPA
- Spring Security
- Gradle 4.10.2
- H2 Database


------

### 개요

- 스프링부트, JPA, 스프링시큐리티를 통한 게시판
- 네이버, 구글, 카카오를 이용한 서드 파티 회원가입 지원
- AWS EC2와 RDS를 사용하여 배포
- 기능별 테스트 코드 작성 및 실행
